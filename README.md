# Hemanth Kumar J - Portfolio Website

A simple, responsive portfolio website showcasing my skills, projects, and experience as a Full Stack Web Developer and AI-Powered Application Builder.

## Features

- Clean, professional design
- Responsive layout for mobile and desktop
- Smooth scrolling navigation
- Sections for About, Education, Skills, Projects, Certifications, and Contact

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a web browser, or
3. Run a local server:
   - If you have Python installed: `python -m http.server 8000`
   - Open `http://localhost:8000` in your browser.
